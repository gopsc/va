# VAB - VA 代理专用浏览器

## 项目简介

VAB 是一个基于 PySide6 的 VA 代理专用浏览器，用于通过 VA 代理服务器访问网页，显示完整的 HTTP 响应信息。

## 功能说明

- **通过 VA 代理服务器访问网页**：所有请求均通过 VA 代理服务器转发
- **显示完整 HTTP 响应信息**：包括状态码、URL、编码、响应头和响应内容
- **支持中文网页解析**：自动检测并使用正确的编码解析中文网页
- **用户友好的界面**：包含地址栏、Go 按钮、Reload 按钮和状态显示
- **不自动加载默认页面**：等待用户输入 URL 后再加载

## 技术栈

- **Python 3.8+**
- **PySide6**：用于构建图形界面
- **requests**：用于发送 HTTP 请求

## 项目结构

```
vab/
├── main.py          # 浏览器主程序
├── requirements.txt # 项目依赖
├── _run.sh         # 运行脚本（Linux）
└── _set.sh         # 设置脚本（Linux）
```

## 安装步骤

1. **安装依赖**

```bash
cd vab
pip install -r requirements.txt
```

2. **启动 VA 代理服务器**

VAB 需要 VA 代理服务器才能工作，请确保 VA 代理服务器已启动。

```bash
# 假设 VA 代理服务器位于 va/src 目录
cd va/src
python main.py
```

## 使用方法

1. **启动 VAB 浏览器**

```bash
cd vab
python main.py
```

2. **访问网页**

- 在地址栏中输入要访问的 URL，例如：`http://qsont.xyz`
- 点击 "Go" 按钮或按回车键
- 浏览器将通过 VA 代理服务器访问该网页，并显示完整的 HTTP 响应信息

3. **刷新页面**

- 点击 "Reload" 按钮可以刷新当前页面

## 响应信息说明

浏览器显示的响应信息包括：

- **状态码**：HTTP 响应状态码，例如 200、404 等
- **URL**：最终访问的 URL（考虑重定向）
- **编码**：网页使用的编码
- **响应头**：完整的 HTTP 响应头
- **响应内容**：网页的 HTML 内容

## 注意事项

1. **VA 代理服务器依赖**：VAB 需要 VA 代理服务器正常运行才能工作
2. **网络连接**：确保设备可以访问 VA 代理服务器（默认端口：7203）
3. **Python 版本**：需要 Python 3.8 或更高版本
4. **权限问题**：在 Linux 系统上，可能需要给脚本添加执行权限：`chmod +x _run.sh _set.sh`
